🎮 Hardware Local: RTX 5090

32 GB VRAM en RTX 5090
405B Parámetros máximos para inferencia
64-128 GB RAM recomendada del sistema
Enero-Febrero 2026

La NVIDIA RTX 5090 se consolida como una de las GPUs más versátiles de 2026 para IA local. Su combinación de densidad de cómputo, eficiencia y precio la hace ideal tanto para ajuste fino local como para inferencia en producción.

Con 32 GB de VRAM, puede manejar modelos de hasta 405B parámetros con técnicas de optimización. Para cargas de trabajo profesionales, se recomienda un mínimo de 64 GB de RAM del sistema, idealmente 128 GB para modelos grandes.

Fuente: Fluence Network, Local AI Master
Recomendación: Para IA local en 2026, la RTX 5090 con 64-128 GB de RAM del sistema es el "sweet spot" entre precio y rendimiento.

🎬 Generación de Video

Sora 2 Líder en fotorrealismo
Runway Gen-4 Kit creativo más completo
Pika 2.5 Mejor para redes sociales
Enero-Marzo 2026

La generación de video por IA ha madurado significativamente en 2026:

  • Sora 2 establece el estándar para calidad cinematográfica y fotorrealismo
  • Runway Gen-4 ofrece el kit de herramientas más completo para profesionales

🎨 Generación de Imágenes

Qwen-Image-2.0 (Alibaba) - ¡Lanzado!

🎉 Lanzado el 10 de Febrero, 2026
10 Febrero 2026

Alibaba lanzó oficialmente Qwen-Image-2.0, un modelo fundacional de generación de imágenes de próxima generación que unifica generación texto-imagen y edición de imágenes en una sola arquitectura.

Características clave:

  • 7B parámetros - arquitectura eficiente
  • Resolución nativa 2K - alta calidad
  • #1 en AI Arena - evaluación ciega líder
  • Excelente en fotos realistas y renderizado de texto complejo
  • Edición integrada - combina Qwen-Image y Qwen-Image-Edit
Fuentes: Qwen.ai, WaveSpeed AI, GIGAZINE

Modelos Open Source 2026

Actualización reciente

Los tres principales modelos open source para aprender en 2026:

  • Stable Diffusion XL - mejor para nuevos usuarios y gran ecosistema
  • Flux 2 - incluye FLUX.2 Klein (distilled 9B & 4B) para generación en tiempo real
  • SD 3 - última iteración de Stability AI (~$0.035/imagen en API)

ComfyUI es la interfaz recomendada para ejecutar modelos open source localmente en 2026.

Fuentes: Pixazo AI, Apatero, BentoML
Precios API 2026: SDXL ~$0.002-0.006/imagen, SD3 ~$0.035/imagen. Qwen-Image-2.0 disponible en Hugging Face.

🔧 llama.cpp - Issues Qwen3.5

⚠️ Estado: Issues abiertos - Sin fixes mergeados confirmados para context caching

Issues Reportados

#19858 - Eval bug: Qwen3.5 always re-processes the full prompt ABIERTO

Qwen3.5 no está reutilizando el contexto, re-procesando el prompt completo en cada solicitud.

#19977 - Qwen3.5-122B-A10B-GGUF loses part of the context cache ABIERTO

Pérdida de parte del cache de contexto con checkpoint invalidado borrado.

#19794 - Qwen3-Coder-Next Prompt Cache forces full re-processing ABIERTO

Cache de prompts fuerza reprocesamiento completo debido a 'invalidated context checkpoint'.

#19690 - Qwen3.5 context caching issues ABIERTO

Problemas generales de caching de contexto en Qwen3.5.

Workaround actual: Usar --ctx-checkpoints 32 y --cache-ram 20000 puede ayudar, pero no resuelve completamente el problema. La comunidad está esperando un fix oficial.

Fix Relacionado

PR #19866 - Qwen 3.5 27B multi-GPU crash fix (mergeado)

Fix para crashes en multi-GPU, pero NO resuelve el issue de context caching.

PR #19849 - Support multi-modal context checkpoints (en progreso)

Podría ayudar con modelos multi-modal, pero aún no está claro si resolverá los issues de Qwen3.5.
